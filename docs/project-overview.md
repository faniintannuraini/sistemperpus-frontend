# Sistem Informasi Perpustakaan Digital

## Deskripsi

Sistem Informasi Perpustakaan Digital merupakan aplikasi berbasis web yang digunakan untuk mengelola proses peminjaman dan pengembalian buku pada perpustakaan Universitas Perjuangan Tasikmalaya.

Sistem memiliki dua jenis pengguna:

1. Mahasiswa
2. Admin Perpustakaan

---

## Fitur Mahasiswa

- Login
- Melihat katalog buku
- Mencari buku
- Filter kategori buku
- Melihat detail buku
- Mengajukan peminjaman buku
- Melihat buku yang sedang dipinjam
- Perpanjang masa pinjam
- Melihat riwayat peminjaman
- Mengelola profil

---

## Fitur Admin

- Dashboard
- Kelola user
- Kelola buku
- Kelola peminjaman
- Kelola pengembalian
- Kelola denda
- Laporan peminjaman
- Laporan denda
- Laporan stok buku
- Kelola profil

## Status Peminjaman

- menunggu_diambil
- sedang_dipinjam
- dikembalikan
- terlambat
